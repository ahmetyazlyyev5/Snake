## Snake Game
A simple Snake game built in Java using the `Swing` framework.

### How to Run Game

```bash
git clone https://github.com/ahmetyazlyyev5/Snake.git
cd Snake
javac -d bin src/*.java
java -cp bin App
```
### Controls
Up: turn up<br />
Down: turn down<br />
Right: turn right<br />
Left: turn left<br />
Spacebar: restart the game<br />
B: toggle gridlines
